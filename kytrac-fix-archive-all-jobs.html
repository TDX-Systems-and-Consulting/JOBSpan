⚠ JOBSpan — Bulk Archive All Jobs

Archives every existing job (soft-delete — same mechanism as the Delete Job button in the app). Archived jobs disappear from the Kanban board, List view, and Home dashboard, but the underlying data (estimates, invoices, change orders, daily logs, everything) stays in Firestore untouched for historical reference. Nothing is permanently deleted. Customers and Vendors are NOT touched.

This affects every job currently in the system — the legacy JobTread/Red Maples import data used to build and validate this app. After running this, your boards and dashboard will show zero active jobs, and only jobs created going forward will count.
