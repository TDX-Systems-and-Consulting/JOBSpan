# JOBSpan

Project Management App

JOBSpan is a project management and operations tracking application built for organizing jobs, budgets, imports, approvals, contracts, statuses, and project workflow activity.

## Current Files

- `index.html` — Main application entry point
- `kytrac-app.js` — Application JavaScript logic
- `kytrac-import.html` — Import page
- `kytrac-import-full.html` — Full import workflow
- `kytrac-migrate.html` — Migration utility
- `kytrac-fix-approved.html` — Approved status utility
- `kytrac-fix-contracts.html` — Contract utility
- `kytrac-fix-estcost.html` — Estimated cost utility
- `kytrac-fix-statuses.html` — Status utility
- `migrate-phases-to-features.js` — Phase-to-feature migration script
